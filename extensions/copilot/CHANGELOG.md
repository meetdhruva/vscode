# Changelog

## 0.47.0

- Rebranded the bundled AI extension shell for Aster.
- Enabled provider-neutral BYOK language model registration without a hosted subscription gate.

- Allowed direct BYOK chat requests without fetching a hosted auth token.

Upstream history from VS Code and the AI extension codebase remains available in git history. Public Aster release notes will track fork-specific changes from this point forward.
